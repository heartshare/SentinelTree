# SentinelTree
阿里Sentinel流量控制技术研究


<pre>
限流：

      当QPS超出某个设定的阈值，系统可以通过拒绝，冷启动，匀速器三种方式来应对，从而起到流量
      控制的作用。
</pre>

<pre>
熔断降级：

      Sentinel通过并发线程数进行限制和响应时间对资源进行降级两种手段来对服务进行熔断或降级。
</pre>

<pre>
塑型：

      通常遇到的流量具有随机性，不规则，不受控的特点，但系统的处理能力往往是有限的，我们需
      要根据系统的处理能力对流量进行塑性，即规则化，从而根据我们的需要来处理流量，
</pre>
